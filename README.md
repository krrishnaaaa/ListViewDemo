# ListView Demo

Demo Android app for creating a ListView.

Companion project for [Create List Using ListView](https://pcsalt.com/android/create-list-using-listview-android/).

## Tech Stack

- Kotlin
- View Binding
- Material 3
- Min SDK 34 / Target SDK 36
